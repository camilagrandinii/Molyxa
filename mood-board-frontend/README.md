<!-- GETTING STARTED -->
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/camilagrandinii/Molyxa
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run app
   ```sh
   npm start
   ```
<p>To add local images, upload them to /public/images and declare them in /src/Data/items.json</p>
